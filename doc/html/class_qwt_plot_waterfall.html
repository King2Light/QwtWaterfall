Detailed Description

Qwt waterfall (datafall) widget.

Note
It is possible to use variables inside QwtWfLayer_t - noscaleX, noscaleY - to set static sizes of the separate layer. Functions append*() are used to append some data to a layer. Colormap is used to map values inside the 2D region to color value.
